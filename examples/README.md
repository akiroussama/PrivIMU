# PrivIMU example CSV files

`demo_motionsense_like.csv` is a small synthetic MotionSense-like CSV used only to test the Streamlit upload UI. It has the same column names expected by the app, including:

- `rotationRate.x`, `rotationRate.y`, `rotationRate.z`
- `userAcceleration.x`, `userAcceleration.y`, `userAcceleration.z`

For real experiments, use the official MotionSense dataset:

```text
https://github.com/mmalekzadeh/motion-sense
https://github.com/mmalekzadeh/motion-sense/raw/master/data/A_DeviceMotion_data.zip
```

After extracting `A_DeviceMotion_data.zip`, upload any file such as:

```text
A_DeviceMotion_data/wlk_7/sub_1.csv
```

The synthetic CSV is not used for final metrics. Final numbers must come from `reports/metrics.json` generated by `python -m privimu.train`.
